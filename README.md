# CesiumClient

Electron-лаунчер для приватного сервера CesiumGDPS (Geometry Dash).

## Возможности

- Автоматическая загрузка и обновление файлов игры
- Проверка целостности файлов (SHA-256)
- Восстановление повреждённых файлов
- Автообновление клиента через GitHub Releases
- Параллельная загрузка (8 потоков) с отображением скорости и ETA

## Сборка

`ash
npm install
npm run build
`

Результат будет в папке dist/.

## Запуск в режиме разработки

`ash
npm start
`

## Иконка приложения

Поместите свой файл icon.ico (содержащий размеры 256x256, 48x48, 32x32, 16x16) в папку uild/.
