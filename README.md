# CesiumGDPSClient.

Официальный клиент CesiumGDPS. Позволяет скачивать актуальную версию игры и обновлять её одной кнопкой.

# Доступные платформы.

Мы стараемся предоставить возможность пользоваться клиентом на каждом устройстве, но это требует времени. Следите за новостями, если ваша платформа не поддерживается.
